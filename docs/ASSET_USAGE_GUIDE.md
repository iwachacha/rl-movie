# RL Movie - AIアセット活用ガイド

このドキュメントは、プロジェクト（特に `Assets/ThirdParty/`）にインポートされているサードパーティ製アセットの特徴、主な収録内容、そしてこれらを使って **AI アシスタント（私）にシナリオのカスタマイズや構築を依頼する際のプロンプトのコツ** をまとめたものです。

## 1. 背景・環境アセット

### Polytope Studio (Lowpoly Environment)
- **収録内容**: 中世ファンタジー風に特化したローポリゴンモデル群。木、草、岩、柵、木箱、樽などの自然物や小さな建物のプレハブが多数含まれます。
- **活用方法**: 強化学習シナリオの「ただの四角い壁と床」を、ゲームのワンシーンのように美しく装飾するために使います。
- **AI への依頼プロンプト例**
  > 「現在の対象シナリオの見た目が少し素朴なので、`Polytope Studio` のアセットから岩や遺跡パーツのプレハブをランダムに配置して、『霧の峡谷』風のステージに装飾してください。」

### BOXOPHOBIC (FREE Skybox Extended Shader)
- **収録内容**: 朝・昼・夕方・夜などを表現するためのキュビマップと、色味や雲の見え方を調整できる Skybox マテリアル。
- **活用方法**: ライティングの印象を大きく変えたい時や、動画全体の雰囲気を一気にリッチにしたい時に使います。
- **AI への依頼プロンプト例**
  > 「シーンの Skybox に `BOXOPHOBIC` の空を適用して、夕方っぽい雰囲気に変更してください。また、太陽光（Directional Light）の色も夕日っぽく調整してください。」

### Fantasy Skybox FREE
- **収録内容**: すぐ差し替えられる複数の skybox / cubemap セット。
- **活用方法**: `BOXOPHOBIC` ほど細かく調整しなくても、シーンの空気感を手早く変えたい時に向いています。
- **AI への依頼プロンプト例**
  > 「`Fantasy Skybox FREE` から明るい昼空の skybox を選んで適用し、現在のステージが見やすくなる方向でライティングも軽く調整してください。」

### Pure Poly / Free Low Poly Nature Pack
- **収録内容**: 森林向けの樹木、草、岩、地形オブジェクトなどの low poly 環境セット。
- **活用方法**: かわいい動物系、配達、障害物回避など、親しみやすい自然環境を作りたい時に相性が良いです。
- **AI への依頼プロンプト例**
  > 「`Pure Poly/Free Low Poly Nature Pack` を使って、木と草の密度を少し増やしつつ、移動経路は読みやすいまま残るように森ステージを装飾してください。」

## 2. キャラクター・動物・アニメーション

### StylizedCharacter (武器付きキャラクター)
- **収録内容**: 剣や盾を持たせることができる、質感の高いデフォルメキャラクターの 3D モデル。武器のモデルも付属しています。
- **活用方法**: ヒューマノイド（人型）エージェントとして利用し、単なるキューブから「戦士」に見た目を変更します。
- **AI への依頼プロンプト例**
  > 「プレイヤーキャラクターを単なる Box から `StylizedCharacter` のモデルに差し替えてください。手に剣の武器モデルを持たせた状態で設定し、Prefab Variant として保存してください。」

### ithappy (Creative Characters & Animals)
- **収録内容**: 犬、猫などの動物モデルや、現代風・ファンタジー風のユニークなローポリアバター。
- **活用方法**: ペンギンや犬などの動物に学習させたい場合や、親しみやすいポップな雰囲気の動画を作りたい時に非常にマッチします。
- **AI への依頼プロンプト例**
  > 「エージェントの見た目を `ithappy` の犬のモデルに差し替えてください。報酬ターゲット側は『骨』のモデル（もしあれば）にするか、色を変えたボールにしてください。」

### Kenney Cube Pets & Googly Eyes
- **収録内容**: 四角くて愛嬌のある様々な動物のモデル（Cube Pets）と、コミカルな目玉のパーツ（Googly Eyes）。すべてCC0の軽量アセット。
- **活用方法**: 学習エージェントやターゲットを「可愛らしいキューブ動物」に差し替える時に最適です。Googly Eyesは、無機質な箱や障害物に貼り付けて「生き物やキャラクター」のように見せるためにも使えます。
- **AI への依頼プロンプト例**
  > 「主人公エージェントのモデルを単なる Box から `Kenney Cube Pets` のアライグマに変更してください。また、向かってくる障害物のキューブには `Kenney Googly Eyes` を貼り付けて顔のようにコミカルにしてください。」

### DoubleL (RPG Animations Pack)
- **収録内容**: アニメーションクリップ（`.anim`）。待機（Idle）、歩行（Walk/Run）、攻撃（Attack）、被弾（Hit）、死亡（Death）、勝利姿勢（One Hand Up）など。
- **活用方法**: アニメーションを含まないモデル（`StylizedCharacter` など）に Animator Controller を設定し、状態に合わせて滑らかに動かすために使います。
- **AI への依頼プロンプト例**
  > 「先ほど設定した `StylizedCharacter` に Animator を設定し、普段は `DoubleL` の Idle を、移動入力が入っているときは Walk アニメーションがブレンドして再生されるようにスクリプトを書いてください。」

### Kevin Iglesias (Human Basic Motions FREE)
- **収録内容**: Human Basic Motions FREE の男性・女性向け基本動作、Avatar Mask、デモ用モデル。
- **活用方法**: 戦闘色を強くしたくない人型シナリオで、待機、歩行、走行、簡易リアクションを入れたい時に向いています。
- **AI への依頼プロンプト例**
  > 「`Creative Characters` ベースのエージェントに、`Kevin Iglesias` の Human Basic Motions FREE から待機と走行を割り当てて、録画時だけ自然に見える Animator を組んでください。」

## 3. エフェクト・パーティクル

### JMO Assets (Cartoon FX Remaster Free)
- **収録内容**: 各種パーティクルシステム（プレハブ）。魔法風の光、爆発、水しぶき、キラキラ系などの派手なエフェクト。
- **活用方法**: エージェントが良いこと（報酬取得）や悪いこと（壁に激突）をした時の**視覚的なフィードバック**として使います。機械学習が「いつ点数を取ったか」を視聴者に一目で伝えやすくなります。
- **AI への依頼プロンプト例**
  > 「エージェントがターゲットに触れてプラスの報酬を得た時に、接触位置に `JMO Assets` (Cartoon FX) の『星が弾けるエフェクト』をインスタンス化するように変更してください。」

### SimpleFX
- **収録内容**: 蒸気、煙、火花、炎、爆発、水しぶきなどの軽量な汎用エフェクト群。
- **活用方法**: 危険地帯、通電床、故障機械などの「ちょっとした迫力」を足したい時に使います。SF 任務系の場面と相性が良いです。
- **AI への依頼プロンプト例**
  > 「`SimpleFX` の蒸気や火花を使って、研究所通路の危険区画に常時ループ演出を追加してください。報酬判定とは切り離し、見た目だけを強化してください。」

### UnityTechnologies (Particle Pack)
- **収録内容**: 炎、火花、煙、爆発などの汎用 particle prefab。
- **活用方法**: `JMO Assets` や `SimpleFX` より少し素直な見た目で、危険表示や軽い衝突演出を足したい時に向いています。
- **AI への依頼プロンプト例**
  > 「`UnityTechnologies/ParticlePack` の火花と煙を使って、通電床や故障機械の近くに小さなループ演出を追加してください。」

## 4. オーディオ / 効果音

### Casual Game Sounds U6 (FREE Casual Game SFX Pack)
- **収録内容**: UI、通知、軽いアクション向けの短い効果音セット。
- **活用方法**: 報酬取得、ターゲット接触、カウントダウンなどを分かりやすく見せたい時に向いています。
- **AI への依頼プロンプト例**
  > 「報酬取得時と失敗時に、`Casual Game Sounds U6` から短い SE を割り当てて、録画で伝わりやすいようにしてください。」

### Free Pack (Free Sound Effects Pack)
- **収録内容**: 汎用 SE、環境音、ループ音、爆発音などを含む大きめの効果音集。
- **活用方法**: `UnityTechnologies/ParticlePack` や `_Tanks` と組み合わせて、環境音や爆発音の厚みを増やしたい時に向いています。
- **AI への依頼プロンプト例**
  > 「`Free Pack` から環境ループ音と爆発音を選び、砂漠シーンの ambient と失敗時の大きめ SE を追加してください。」

## 5. アニメーション制御（UI・カメラ・ギミック）

### PrimeTween / Demigiant (DOTween)
- **収録内容**: スクリプトから簡単に、しかも高パフォーマンスで「A から B へ移動する」「徐々に透明になる」「揺れる」といった処理を書けるライブラリ API。
- **活用方法**: Unity の Animator を使わずに動く障害物を作ったり、UI を気持ちよく表示させたり、ダメージ時にカメラを揺らす等に使用します。本プロジェクトではより高速な **PrimeTween** を推奨します。
- **AI への依頼プロンプト例**
  > 「ステージ中央の障害物を、`PrimeTween` を使って Z 軸方向に 5 の距離を 2 秒かけて滑らかに往復移動するギミックとして実装してください。」
  >
  > 「エージェントがマイナス報酬を受けた際に、メインカメラが `PrimeTween` を使って 0.2 秒間だけ小さく揺れる Camera Shake 演出を追加してください。」

## 6. 参考プロジェクト（対戦・射撃）

### _Tanks (Tanks! Complete Project)
- **収録内容**: Unity 公式の対戦型戦車ゲームの全アセット。戦車モデル、大砲の弾、爆発エフェクト、エンジン音・発射音の AudioClip、およびそれらを動かす完成済みスクリプト群。
- **活用方法**: 「弾を撃って相手に当てる」「HP がゼロになったら壊れる」といったロジックを、強化学習シナリオの環境ルールとして導入したい場合に、ここのコードや効果音を**参考・流用**します。
- **AI への依頼プロンプト例**
  > 「射撃行動を学習するシナリオを作りたいです。`_Tanks` プロジェクトの中にある大砲のプレハブと発射音（Audio）を流用して、エージェントがスペースキーで弾を撃てるように `TankRLAgent` スクリプトを組んでください。」

## 7. 組み合わせのコツ

- `Polytope Studio` + `StylizedCharacter` + `DoubleL` + `JMO Assets`
  ファンタジー、遺跡、剣士、儀式、戦闘風の見せ場に向きます。
- `ithappy/Animals_FREE` + `Polytope Studio` + `PrimeTween`
  かわいい動物系、障害物回避、配達、追いかけっこ系に向きます。
- `Pure Poly/Free Low Poly Nature Pack` + `ithappy/Animals_FREE` + `Casual Game Sounds U6`
  森や自然環境の癒やし寄りシーンづくりに向きます。
- `Free Pack` + `_Tanks` + `JMO Assets`
  爆発音、重めの SE、派手なヒット演出をまとめて強化したい時に向きます。

## 8. 私（AI エージェント）に指示を出す際のコツ

1. **「アセット名」と「目的」をセットにする**
   「`JMO Assets` を使って豪華にして」より、「**報酬を獲得した時** に `JMO Assets` を使って **分かりやすいエフェクト** を追加して」のように目的も添えてもらえると、タイミングまで含めて実装しやすくなります。

2. **Prefab Variant を活用した安全な実装にする**
   ルール（`ASSET_CATALOG.md`）に従い、ThirdParty 内の元アセットは直接編集せず、シナリオフォルダ内に `Prefab Variant` を作成して機能を追加します。「Prefab Variant を作って使って」と言ってもらえると、その前提で安全に進められます。

3. **ざっくりした依頼でも大丈夫**
   「`StylizedCharacter` を使って、敵から逃げ回る見栄えの良いアクションゲーム風の回避学習シナリオを作って」のような大きめの依頼でも、このドキュメントの内容に基づいてアセットの組み合わせや実装プランを提案できます。
