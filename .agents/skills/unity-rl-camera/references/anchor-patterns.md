# Anchor Patterns

Use this file to choose anchor geometry based on the scene layout.
Treat these as starting patterns, then adjust for the actual goal, hazard, and motion.

## Arena

Best for:
- contained arenas
- dodge, push, survive, or timing tasks

Start with:
- `Explain`: high 3/4 overview
- `Wide A`: left diagonal
- `Wide B`: right diagonal
- `Follow`: rear or rear-quarter if motion quality matters

Avoid:
- perfectly top-down shots that flatten motion
- low angles that hide the arena layout

## Course

Best for:
- obstacle lanes
- traversal or chase paths
- tasks with strong forward progress

Start with:
- `Explain`: high oblique showing the next section of course
- `Wide A`: broad side or quarter-side cut
- `Wide B`: reverse quarter or goal-side cut
- `Follow`: trailing shot only if upcoming obstacles are still readable

Avoid:
- follow shots that hide what is coming next
- frontal shots that remove depth cues

## Room

Best for:
- enclosed spaces
- labs, chambers, or tight interior puzzles

Start with:
- `Explain`: corner-high diagonal with two walls readable
- `Wide A`: opposite or adjacent diagonal
- `Wide B`: side cut only if it clarifies failure causes
- `Follow`: use only when occlusion is manageable or ghosting helps

Avoid:
- anchors pressed against walls
- angles that let ceilings or door frames dominate the frame

## Field

Best for:
- open zones
- larger navigation spaces

Start with:
- `Explain`: elevated oblique with enough horizon or boundary context
- `Wide A`: closer diagonal toward the action cluster
- `Wide B`: opposing diagonal or a safer side cut
- `Follow`: only when the field is not so large that the target disappears

Avoid:
- placing the action as a tiny speck in a huge landscape
- cutting so close that the field stops being understandable

## Swarm Or Group Behavior

Best for:
- many-agent formations
- flocking, avoidance, battle lines, or crowd comedy

Start with:
- `Explain`: wider and higher than usual so the pattern reads
- `Wide A`: angle that shows the shape of the group
- `Wide B`: angle that shows collisions, splits, or regrouping
- `Follow`: optional, usually one brief personality cut on a focal agent

Avoid:
- hero-shot logic when the spectacle is actually the whole pattern
- staying so close that the swarm shape disappears

## Red Flags

- one angle that looks nice but fails to explain the space
- mirrored wides with no informational difference
- a follow shot with no stable wide to anchor it
- anchors that ignore where failure actually happens
