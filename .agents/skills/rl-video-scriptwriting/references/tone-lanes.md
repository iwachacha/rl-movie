# Tone Lanes

Choose one lane on purpose.
Borrow the structural strengths of successful channels without copying their exact persona.

## Chaotic Comedy

Best for:
- slapstick failure
- tiny agent versus huge danger
- absurd physics or greedy mistakes

Voice shape:
- short
- punchy
- slightly incredulous

Use this lane when failure is the main entertainment.
Avoid it when the footage is calm, elegant, or uncanny rather than funny.

## Tense Challenge

Best for:
- timers
- hazards
- near-misses
- tight escapes

Voice shape:
- direct
- pressure-heavy
- low on jokes

Use this lane when the viewer should feel urgency.
Keep lines sparse so the danger still breathes.

## Curious Documentary

Best for:
- readable systems
- multi-step behavior
- calm but interesting competence
- emergent interactions

Voice shape:
- observational
- dry
- lightly amused

Use this lane when the video should feel guided rather than hyped.
Avoid overexplaining; the strength is calm confidence.

## Uncanny Competence

Best for:
- eerie progress
- smooth exploit-seeking
- behavior that feels smarter than it really is

Voice shape:
- sparse
- slightly ominous
- controlled

Use this lane when the agent becomes unsettling or strangely efficient.
Avoid constant jokes that would break the mood.

## Selection Rule

- If failure is the hook, start with `Chaotic comedy`.
- If danger is the hook, start with `Tense challenge`.
- If system behavior is the hook, start with `Curious documentary`.
- If the solve feels weirdly smart, start with `Uncanny competence`.
