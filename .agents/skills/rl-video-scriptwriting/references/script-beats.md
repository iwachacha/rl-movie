# Script Beats

Use narration only where it improves clarity, tension, comedy, or pacing.
The default assumption for this project is a short-form or tightly edited RL video.

## Default Beat Map

1. Cold open
2. Rule or premise
3. Early chaos
4. Breakthrough
5. Payoff
6. Optional button

## Cold Open

Use zero to two short lines.
The job is to create curiosity fast, not to explain everything.

Good jobs:
- tease the weirdest failure
- show the danger in one sentence
- ask the question the rest of the video answers

Bad jobs:
- full scenario explanation
- algorithm setup
- backstory before the viewer sees motion

## Rule Or Premise

Keep this to one sentence or one short text card when possible.
State the goal, the hazard, or the success condition.

Good:
- say what must happen
- say what can go wrong

Bad:
- list observations or actions
- explain reward shaping

## Early Chaos

Use short lines only on distinct failures.
The job is to show that the agent is not there yet, while keeping momentum.

Good jobs:
- label a recurring failure pattern
- escalate from dumb to almost-smart
- bridge a time jump between training stages

Bad jobs:
- narrate every attempt
- explain why each miss matters

## Breakthrough

This is where the viewer first feels a real behavioral leap.
Let the change become visible, then name it.

Good jobs:
- point out the new strategy
- frame the exploit or clever shortcut
- mark the first moment of real competence

## Payoff

Often this beat needs little or no narration.
If you speak here, use one line that lands after the action, not before it.

Good jobs:
- answer the cold-open question
- underline how clean the solve became
- leave one final emotional note

## Optional Button

Use only when it adds something new.

Good endings:
- one last funny fail after success
- a quiet flex shot
- a comparison beat on matching framing

Bad endings:
- several redundant wins
- a metrics screen

## Silence Rules

- If the viewer can instantly read the action, consider silence.
- Leave breathing room after a huge fail or a clean solve.
- Do not fear a silent beat if the footage already carries the joke.
