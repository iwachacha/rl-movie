## 現在の入口
- repo を調べる・変更する依頼では、先に `.agents/core/SKILL.md` を読む
- その後は `.agents/skills/` 配下から一致する skill だけを読む
- `.agents/workflows/` は legacy alias / checklist であり、固有ルールの正本ではない
- 軽い非 repo 質問では `.agents` を追加読込しない
- `docs/` は人向け補助資料で、必要なときだけ参照する
